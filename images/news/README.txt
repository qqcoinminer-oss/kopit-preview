KOPIT NEWS IMAGES — UPLOAD GUIDE

Place all news article images in this folder.

NAMING:
- Use lowercase letters, numbers, and hyphens only
- Examples: k3-pods.jpg, event-cover.png, q3-report.jpg
- AVOID: spaces, uppercase, special characters

RECOMMENDED SIZES:
- Cover image (featured article): 1200 x 800 pixels
- Cover image (standard article): 800 x 500 pixels
- Gallery thumbnails: any size (auto-cropped to square)
- Banner cover (4+ image articles): 1200 x 600 pixels

FILE FORMATS: .jpg, .png, .webp
FILE SIZE: Keep under 500KB per image for fast loading

REFERENCE IN ARTICLES.JSON:
"images": ["k3-pods.jpg", "k3-water-test.jpg", "k3-team.jpg"]

The first image in the array is always the cover.
Add up to 9 images per article (first = cover, others = gallery).
